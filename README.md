# Code-Jump-Pau-s-Sky-Mission
GUI PROJECT - OOP

## Description
CodeJump: Pau's Sky Mission is a 2D platformer game developed using Python and Pygame. The player collects coins, jumps across platforms, and avoids falling while aiming for the highest score.

## Features
- Main menu
- Player name input
- Coin collection
- Leaderboard
- Game Over screen
- Background music and sound effects

## Technologies Used
- Python
- Pygame

## How to Run
1. Install Python.
2. Install Pygame:
   ```
   pip install pygame
   ```
3. Run the game:
   ```
   python main.py
   ```
